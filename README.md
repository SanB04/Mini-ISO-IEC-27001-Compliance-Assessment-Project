**Project Overview:**
This project demonstrates a mini implementation of an Information Security Management System (ISMS) aligned with ISO/IEC 27001 for a fictional SaaS startup named Nova Cloud Analytics Pvt. Ltd.
The objective was to select and map required controls from Annex A, evaluate whether the controls have already been implemented, partially implemented or not yet implemented. Gaps were identified that needed to be worked on.

**Project Activities:**
• Defined the ISMS scope for Nova Cloud Analytics.
• Identified and documented critical assets including cloud infrastructure, intellectual properties and workstations.
• Created an asset inventory that contains important assets, ownership, classification, and system dependencies.
• Conducted compliance assessmnet by selecting and mapping required controls from Annex A.
•Analysed gaps that needed to be worked on.

**Key Deliverables:**
• ISMS Scope Document
• Asset Inventory Spreadsheet
• Compliance Assessment Spreadsheet
• Statement of Applicability
